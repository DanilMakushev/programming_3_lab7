# Поиск файлов-дубликатов

## Параметры командной строки
- `-d <dir>` Директория для сканирования (если несколько, то перед каждой добавлять `-d`)
- `-e <dir>` Исключить директорию из сканирования (если несколько, то перед каждой добавлять `-e`)
- `-l <n>` Глубина сканирования: `0` - только сама директория (без вложенных), `1` - включая вложенные
- `-m <bytes>` Минимальный размер файла (по умолчанию 1 байт)
- `-p <mask>` Маски файлов (*.txt, *.cpp, можно несколько через `;`)
- `-s <bytes>` Размер блока
